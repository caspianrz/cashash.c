# cashash.c

`cashash.c` is a small hash table library written in C.

It provides a simple string-key hash table with generic pointer values. The library is designed to be minimal, portable, and easy to embed in C projects.

## Features

- String keys
- Generic `void *` values
- Separate chaining
- FNV-1a hashing
- Dynamic bucket count
- Static library builds
- Cross-platform support for Linux, macOS, and Windows

## Documentation

API documentation is generated with Doxygen and published through GitHub Pages.

## Testing

The project uses the Check unit testing framework and is tested through GitHub Actions across multiple platforms.
